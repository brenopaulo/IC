# Inteligência Computacional

Inteligência Computacional é uma disciplina do 7º semestre de Engenharia da Computação da UFPA e possui uma carga horária de 60 horas.

# Projetos

Durante a disciplina, foram desenvolvidos 3 projetos:

## LLM_RAG

Este projeto apresenta uma análise comparativa do desempenho de um modelo de linguagem (LLM) operando com e sem a técnica de Geração Aumentada de Recuperação (RAG). Utilizando o Código de Ética Profissional do CONFEA como base de conhecimento, o estudo submete ambos os cenários a um conjunto de perguntas preestabelecidas para avaliar e contrastar a precisão e a fundamentação das respostas obtidas.

## Relatório_IC___Tarefa_de_Aprendizado_por_Reforço__RL_

Este projeto teve como objetivo explorar a aplicação prática de Aprendizado por Reforço (RL) através da comparação de desempenho entre os algoritmos PPO (Proximal Policy Optimization) e A2C (Advantage Actor-Critic). O estudo submeteu ambos os agentes ao ambiente ALE/MarioBros-v5. O trabalho foi fundamental para observar como diferentes arquiteturas de redes neurais lidam com entradas de pixels e planejamento de longo prazo em emuladores de Atari.

## Relatório_IC___Projeto_Final

Este projeto investigou a sensibilidade do algoritmo PPO a variações críticas nos hiperparâmetros de Taxa de Aprendizado (Learning Rate) e Fator de Desconto (Gamma). A pesquisa utilizou o ambiente Mountain Car Continuous-v0, um cenário onde o agente deve aprender a física de oscilação para superar a gravidade e atingir o topo de uma montanha.
